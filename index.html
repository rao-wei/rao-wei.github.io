vue生命周期(钩子函数)

  1. create - mount - update - destroy 四个节点 每个节点有前后
  2. beforeCreate - created
  3. beforeMount - mounted
  4. beforeUpdate - updated
  5. beforeDestroy - destroyed()

vue开发环境安装配置

  1. node.js安装
  2. 设置淘宝镜像 npm config set registry https://registry.npm.taobao.org
  3. 安装vue脚手架 npm install -g vue-cli
  4. 新建vue项目 vue init webpack name
  5. 进入文件夹 把配置文件的插件安装一遍 npm i
  6. 运行开发环境 npm run dev / npm start
  7. 把项目编译成可以上线的静态html css js npm run build

vue路由常用功能

  1. 路由跳转

    在ruoter下面的index.js中配置

    先将需要跳转的页面导入依赖 import index from '@/components/index' 其中@代表的是src这个目录

    然后再在export default new Router里面配置路径

    const router = new Router({
    routes: [

    {
  path: '/',
  name: 'index',
  component: index,
  beforeEnter: (to, from, next) => {
    // console.log('我是路由配置里面的路由钩子函数-即将进入首页之前触发',to,from)
    next()
  }
}

    ]
    })

    router.beforeEach(async (to, from, next) => {
    console.log(‘我是路由全局前置钩子函数-每次切换路由之前都触发’,to,from)
    // if(to.path != ‘/signin’){
    // let res = await axios.get(‘/static/check.json’)
    // res = res.data
    // if(res == 1){
    // next()
    // }else{
    // router.push(‘/signin’)
    // }
    // }else{
    // next()
    // }
    next()
    })
    `

    路由全局后置钩子函数
