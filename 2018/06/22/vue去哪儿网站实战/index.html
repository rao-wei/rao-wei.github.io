vue去哪儿网站实战

发表于 2018-06-22   |   阅读次数

使用vue开发去哪网

设计图

依赖包下载：

npm install stylus --save

npm install stylus-loader --save

组件开发：

header.vue组件开发

  1. 在components下面添加header.vue页面
  2. 在Home.vue上引入header.vue import HomeHeader from './components/Header'
  3. components 上添加HomeHeader组件

header分为三个部分 left input right

iconfont的使用

在阿里巴巴矢量图库下载代码，将代码文件中的字体文件保存到assets的styles中新建一个iconfont放字体文件，iconfont.css放在styles中，将iconfont.css中的字体地址更改前面加上./icoonfont

在main.js中导入iconfont.css

import '@/assets/styles/iconfont.css'

在需要使用的html文档中给包裹图片的div或span等标签加上iconfont类名内容写对应图标的代码就可以<span>&#xe600;</span>

使用stylus设置css变量

在styles文件夹下新建一个varibles.styl的全局stylus文件

在需要使用的style中引入@import '~styles/varibles.styl'

提示：styles是在build下面的webpack.base文件下设置的快捷地址

在resolve中新增 'styles': resolve('src/assets/styles')

在varibles.styl中定义变量,style中就可以直接使用这个变量了

\$bgColor = #00bcd4
